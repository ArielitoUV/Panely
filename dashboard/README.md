# PANADERÍA — Cliente (Next.js)

Bienvenido al cliente del proyecto PANADERÍA. Este README explica cómo instalar las dependencias, ejecutar el proyecto en modo desarrollo y generar una build de producción de forma sencilla.

## Requisitos

- Node.js (recomendado >= 16, <= 20)
- npm (v8+)

## Ejecución rápida

1. Instalar dependencias:

```bash
npm i --legacy-peer-deps
```

2. Levantar el servidor en modo desarrollo:

```bash
npm run dev
```

Abre tu navegador en http://localhost:3000 (o el puerto que indique la consola).

3. Generar build de producción:

```bash
npm run build
```

4. (Opcional) Probar la build localmente:

```bash
npm run start
```
